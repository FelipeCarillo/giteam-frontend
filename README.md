# GitTeams Frontend 🚀

Interface web moderna e responsiva para o GitTeams - plataforma de agentes de IA para GitHub, construída com React e Material-UI.

## ✨ Características

- **🎨 Design Moderno**: Interface clean com suporte a tema escuro/claro
- **🌍 Multilíngue**: Suporte completo para Português e Inglês
- **📱 Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **🔒 Autenticação Segura**: Login via GitHub OAuth
- **⚡ Performance**: Otimizado com lazy loading e cache inteligente
- **🎯 UX Intuitiva**: Navegação fluida e feedback visual em tempo real

## 🛠️ Tecnologias

- **React 18**: Framework principal com hooks modernos
- **Material-UI (MUI) 5**: Biblioteca de componentes UI
- **React Router**: Navegação SPA
- **Axios**: Cliente HTTP para APIs
- **Context API**: Gerenciamento de estado global
- **React Hooks**: useState, useEffect, useContext, etc.

## 📋 Pré-requisitos

- Node.js 18+ 
- npm ou yarn
- GitTeams Backend rodando
- Conta no GitHub para autenticação
